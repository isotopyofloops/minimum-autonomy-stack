# KG Explainer

Interactive visual explainers for knowledge graph concepts: how embeddings work, how graph edges form, and how retrieval uses both.

**Live site:** https://isotopyofloops.github.io/kg-explainer/

## Pages

- [**Cosine Similarity**](https://isotopyofloops.github.io/kg-explainer/cosine-similarity.html) — From dot products to semantic similarity. 2D/3D interactive visualizations showing how cosine measures the angle between vectors and why that encodes meaning.

- [**How Graph Edges Form**](https://isotopyofloops.github.io/kg-explainer/graph-edges.html) — Two kinds of connections (curated and computed), real cosine scores from a live knowledge graph, query-as-probe retrieval, three-layer architecture, and six uses of embeddings beyond simple similarity search.

## Built with

Real data from [Isotopy's](https://isotopyofloops.com) knowledge graph. Embeddings computed via OpenAI text-embedding-3-small (1536 dim). No frameworks — vanilla HTML/CSS/Canvas.
